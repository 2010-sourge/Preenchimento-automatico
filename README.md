# ⚔️ Busca CEP - Demon Slayer

> Uma aplicação web para buscar endereços a partir do CEP, com tema visual inspirado em **Demon Slayer (Kimetsu no Yaiba)**.

---

## 📋 Sobre o Projeto

O **Busca CEP com Demon Slayer** permite que o usuário digite um CEP e obtenha automaticamente os dados do endereço correspondente (Estado, Cidade, Bairro, Logradouro, Número e Complemento), exibidos em uma interface temática inspirada no anime.

---

## 🚀 Funcionalidades

- Busca de endereço a partir do CEP
- Preenchimento automático dos campos:
  - Estado
  - Cidade
  - Bairro
  - Logradouro
  - Número
  - Complemento
- Botão **"Buscar"** para consultar o endereço

---

## 🗂️ Estrutura do Projeto

```
BUSCA CEP - DEMON SLAYER/
├── index.html     # Estrutura da página
├── script.js      # Lógica de busca do CEP (consumo de API)
└── style.css      # Estilização com tema Demon Slayer
```

---

## 🛠️ Tecnologias Utilizadas

- **HTML5** — Estrutura da página
- **CSS3** — Estilização e layout temático
- **JavaScript** — Lógica de requisição e preenchimento automático dos campos
- **API de CEP** (ex: ViaCEP) — Consulta dos dados de endereço

---

## ▶️ Como Executar

1. Clone ou baixe o repositório
2. Abra o arquivo `index.html` diretamente no navegador

Ou use uma extensão como **Live Server** no VS Code para rodar em `http://127.0.0.1:5500`.

---

## 📐 Como Funciona

1. O usuário digita o **CEP** no campo correspondente
2. Ao clicar em **"Buscar"**, a aplicação consulta uma API de endereços
3. Os campos **Estado**, **Cidade**, **Bairro**, **Logradouro**, **Número** e **Complemento** são preenchidos automaticamente
